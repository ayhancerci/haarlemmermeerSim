# Project Roadmap

## Phase 1: MVP (Current) — Emergency Preparedness Game
- [x] 4 emergency scenarios (Flooding, Industrial Accident, Extreme Weather, Infrastructure Failure)
- [x] 3-phase gameplay loop (Preparation → Emergency → Recovery)
- [x] Interactive CesiumJS map with real Haarlemmermeer POI locations
- [x] 5 languages (NL, EN, TR, AR, UK)
- [x] Decision-based learning (11 decisions per scenario)
- [x] Emergency kit packing mini-game
- [x] Knowledge quiz (7 questions per scenario)
- [x] Personalised debrief with score feedback
- [x] Mobile responsive design
- [x] Static deployment (GitHub Pages / any web server)

## Phase 2: Enhanced Experience
- [ ] Interactive tutorial for first-time users
- [ ] Flood zone visualization (progressive water rise by postcode elevation)
- [ ] Toxic plume animation for Industrial Accident scenario
- [ ] Storm intensity overlay for Extreme Weather scenario
- [ ] Power outage cascade visualization for Infrastructure Failure
- [ ] Back button from Briefing to Scenario Select
- [ ] Haptic feedback on mobile (vibration on decisions/alerts)
- [ ] Sound effects (NL-Alert tone, radio jingle, timer warning)
- [ ] Accessibility improvements (WCAG AA contrast, screen reader support)
- [ ] Quiz answer randomization

## Phase 3: Community & Sharing
- [ ] Share score on social media (LinkedIn, Twitter, Facebook)
- [ ] Generate PDF certificate of completion
- [ ] Leaderboard (local storage or optional server)
- [ ] QR code for easy sharing at community events
- [ ] Embed mode (iframe for municipality websites)
- [ ] Print-friendly emergency checklist export

## Phase 4: Advanced Features
- [ ] Multiplayer mode (family plays together, each member has a role)
- [ ] Custom scenario builder (municipality can create scenarios)
- [ ] SCORM package export (for LMS integration)
- [ ] Analytics dashboard (anonymous usage data for municipality)
- [ ] AI-powered dynamic decisions (personalised based on player behavior)
- [ ] Integration with real-time weather data (KNMI API)
- [ ] Integration with real NL-Alert test broadcasts
- [ ] VR/AR mode for immersive training

## Phase 5: Platform & Scale
- [ ] SaaS platform for other municipalities
- [ ] White-label version for Veiligheidsregio's
- [ ] EU compliance (GDPR, accessibility directive)
- [ ] Multi-country support (Belgium, Germany, UK emergency protocols)
- [ ] n8n workflow integration for automated SCORM generation
- [ ] API for third-party integrations
